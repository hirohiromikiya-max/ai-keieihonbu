## 📌 予約仕入れの基本概念

### 通常仕入れとの違い

| 項目 | 通常仕入れ | 予約仕入れ |
|------|----------|----------|
| 仕入れタイミング | 発売後・在庫あり | **発売前・予約注文** |
| 価格 | 定価〜プレミア価格 | **定価（最安値）** |
| 在庫リスク | 低い | やや高い（需要予測必要） |
| 利益率 | 中〜高 | **非常に高い（5〜15倍も可）** |
| 競合 | 多い | **少ない（発売前は希少）** |

### 予約仕入れが強い理由
- 定価で仕入れられる唯一のタイミング
- 発売前から出品→海外バイヤーの予約需要を取れる
- 発売後に在庫切れになれば価格が上がる

## 🎯 予約仕入れ対象商品の選定基準

### 対象ジャンル
- **ジャンプ系雑誌**（最強ジャンプ・週刊少年ジャンプ付録・JUMP GIGA等）
- **ベイブレードX**（新弾・コラボ商品）
- **限定・コラボ商品**（エヴァコラボ等）
- **応募者全員プレゼント・付録カード**

### GO基準（全て満たすこと）

| 条件 | 基準 |
|------|------|
| ① eBay需要確認 | 類似商品・前号がeBayで売れているか |
| ② 価格差 | 定価仕入れ→eBay販売で利益率25%以上 |
| ③ 希少性 | 日本限定・応募者限定・コラボ等 |
| ④ 仕入れ可能 | Amazon・あみあみ等で定価予約可能 |

## 🔍 STEP 1：需要リサーチ

### 1-1. eBayで類似商品・前号を検索
1. eBayで「商品名 + Japan」で検索
2. **Sold**タブで過去の売れ値を確認
3. 確認ポイント：
   - いくらで売れているか？
   - 何件売れているか？
   - 予約出品（PREORDER）で売れているか？

### 1-2. 予約販売の需要確認
1. eBayで「PREORDER + 商品名」で検索
2. 既に出品されているか・ウォッチャー数を確認
3. 競合出品者が少なければチャンス✅

### 1-3. 判定基準

| 状況 | 判定 |
|------|------|
| 前号・類似商品が定価の3倍以上で売れている | 即予約GO✅ |
| 前号が定価の1.5〜3倍で売れている | 数量限定で予約✅ |
| 前号がほぼ定価で売れている | 見送り❌ |
| 予約出品のウォッチャーが5人以上 | 需要あり✅ |

## 📦 STEP 2：仕入れ実行

### 仕入れ先（定価購入）

| 仕入れ先 | 対象商品 | 特徴 |
|---------|---------|------|
| Amazon.co.jp | ジャンプ系雑誌・書籍 | 即購入・確実 |
| あみあみ | フィギュア・ベイブレード | 予約キャンセル可 |
| 駿河屋 | ゲーム・フィギュア | ポイント活用可 |
| 楽天市場 | 各種グッズ | ポイント活用可 |
| セブンネット | 雑誌・書籍 | コンビニ受取可 |

### 仕入れ数量の目安

| 需要レベル | 仕入れ数 |
|-----------|---------|
| 高（前号3倍以上売れ） | 3〜5個 |
| 中（前号1.5〜3倍売れ） | 1〜2個 |
| 初回テスト | 1個から |

---

## 💰 STEP 3：価格設定

### 予約出品価格の考え方
- **発売前（予約期間）：** 強気価格（競合少ないため）
- **発売直後：** 市場価格を確認して調整
- **在庫残り時：** 利益率25%を下限に値下げ

### 計算例（最強ジャンプ応募者全員プレゼントカード）
- 仕入れ値：¥664（Amazon定価）
- eBay出品価格：EUR54.90（約¥10,000）
- 利益率：**約93%** → 超優良案件✅

### 価格設定ルール
1. 在庫管理シートI列（25%最低価格）で下限を確認
2. eBayの競合最安値を確認
3. **競合より10〜20%高くても売れる**（希少性があれば）
4. 発売後2週間で売れなければ価格を見直す

## ⚠️ リスク管理

### 売れなかった場合の対処
1. 価格を下げてeBayで継続販売
2. メルカリ・ヤフオクで国内販売
3. 定価割れは原則しない（損切りライン：仕入れ値の80%）

### 予約キャンセルポリシー確認
- あみあみ：発売〇日前までキャンセル可
- Amazon：発売前はキャンセル可（要確認）
- 需要が見込めない場合は早めにキャンセル
