| 分類 | 目標比率 | 特徴 |
| :---- | :---- | :---- |

## 🔄 回転商品リサーチ（既存ルール）

### 選定基準

- eBay Product Research（anime / LP / manga 等）  
- Seller's location：Japan  
- 期間：7日間  
- **7日間で5個以上**売れている商品

### 価格設定

- 日本発送セラーの**最安値3位以内**

## 🟢 ロングテール通年型リサーチ

### 選定基準（4条件すべて満たすこと）

| 条件 | 基準 |
| :---- | :---- |
| ① 売れ頻度 | 年間6〜36個（月0.5〜3個ペース） |
| ② 売れ方 | 特定季節に偏っていない・通年で分散している |
| ③ 競合数 | 日本発送セラーが5件以下 |
| ④ 利益率 | 25%以上（在庫管理シートI列基準） |

### 調査手順

**STEP 1：starfish\_japanのSold確認**

1. starfish\_japanのSold一覧を開く  
2. 期間を**365日**に設定  
3. 年間6〜36個売れている商品をピックアップ

**STEP 2：売れ方の分散確認**

1. eBay Product Researchで商品名キーワード検索  
2. 期間：**365日**  
3. グラフで月別売れ数を確認  
4. 判定：  
   - 毎月コンスタントに売れている → 通年型✅  
   - 特定月に偏っている → 季節型に分類🌸  
   - ほぼ売れていない → 見送り❌

**STEP 3：競合確認**

1. eBayで同キーワード検索  
2. フィルター：Ship from Japan・Buy It Now  
3. 競合5件以下 → 候補✅

**STEP 4：利益計算**

- 在庫管理シートI列（25%最低価格）で採算確認  
- 競合最安値3位以内で利益率25%以上 → 仕入れGO✅

## 🌸 ロングテール季節型リサーチ

### 選定基準

| 条件 | 基準 |
| :---- | :---- |
| ① 売れ頻度 | 特定の季節（1〜3ヶ月）に集中して売れる |
| ② ピーク月 | 年間売上の50%以上が特定月に集中 |
| ③ 競合数 | 日本発送セラーが5件以下 |
| ④ 利益率 | 25%以上 |

### 季節パターン例

| 季節 | 対象商品例 |
| :---- | :---- |
| 年末（11〜12月） | おもちゃ・フィギュア・ゲーム |
| 春（3〜4月） | 卒業・入学関連・書籍・漫画 |
| 映画・アニメ公開時期 | 関連グッズ・限定品 |
| バレンタイン（2月） | キャラクターグッズ |

### 運用ルール

- **ピーク月の2〜3ヶ月前**に仕入れ・出品開始  
- ピーク終了後は価格を下げて在庫を消化  
- 翌年の仕入れ時期をメモに残す

### 調査手順

**STEP 1：季節性の確認**

1. eBay Product Researchで対象商品を検索  
2. 期間：**365日**  
3. グラフで月別売れ数を確認  
4. 特定月に山がある → 季節型✅

**STEP 2：ピーク月の特定**

- グラフから売れピーク月を記録  
- 「いつ仕入れるか」をメモに残す

**STEP 3：競合・利益確認**

- 通年型と同じ手順で確認

## ⚠️ 共通：見送り基準

- 年間売れ数が6個未満  
- 競合が10件以上  
- 利益率25%未満  
- 仕入れ値が高すぎて価格競争できない  
- 送料コストが重量オーバーで採算割れ
